Transfinder Form Assistant Design

An AI-powered system for generating forms, processing complaints, and creating student reports through natural language interaction

Project Overview

The Transfinder Form Assistant is a comprehensive system that leverages AI to help users create forms, file complaints, and generate reports through conversational interfaces. The system integrates with Ground Control for essential AI interaction components.

Key Features

  • Complaint Management: Multi-phase dialogue system for filing and processing complaints against students
  • Form Builder: AI-generated form JSON and HTML pages with dark gray and orange theme
  • Report Generator: SQL query generation and execution for student data reports
  • Chat Interface: Natural language processing for all interactions
  • Voice Support: Voice input and recognition capabilities
  • State Management: Persistent session state using BadgerDB

System Architecture

The system is built with a Go backend and React frontend, using:

  • Backend: Go (Gin framework) with BadgerDB for storage
  • Frontend: React with modern chat interface
  • AI Integration: Qwen3-Max model via DashScope API
  • Database: BadgerDB (key-value store) + SQL Server for student data
  • External Services: Ground Control API for dialogue-driven flows
